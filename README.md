# Skills-and-Practices-Framework
This is an open source information architecture built and maintained by ProblemOps Institute https://problemops.com. This framework is an architecture of information that describes everything it takes to succeed in team-based problem-solving. It shows the relational data that is involved and examples of common data. Organizations and people can use this data to build training and deliverables around project management, product management, and service delivery for any kind of industry.

# API Documentation
Here's the API documentation for the database hosted in Baserow: https://baserow.io/api-docs/database/346422

Using this API documentation, people can utilize the database data that exists for their own development needs.

# Key Use Cases
- Understanding job requirements
- Measuring success in skill development
- Evaluating team performance
- Hiring the right person for a role
- Defining job requirements based on industry standards
